DMLite tutorial

Requirements

Installing the dmlite-devel RPM should push all the necessary build dependencies. You probably will need some plugins installed to do the tests, as dmlite-plugins-mysql, for instance, and configure them as explained in the configuration guide.

The compilation commands shown assume that you have these components in their standard locations, and that the version of the Boost libraries is equal or higher than 1.41.
For SL5, you can install boost141-devel from EPEL, but in that case you will need to add the parameter -I /usr/include/boost141
